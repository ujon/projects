# 개인정보처리방침

**최종 수정일: 2024년 12월 19일**

## 개요

Timepaw는 사용자의 개인정보 보호를 최우선으로 합니다. 본 개인정보처리방침은 타임스탬프 기록 애플리케이션에서 정보를 어떻게 처리하는지 설명합니다.

## 수집하는 정보

**당사는 어떠한 개인정보도 수집하지 않습니다.**

TimePaw는 프라이버시를 핵심 원칙으로 설계되었습니다. 사용자가 생성한 모든 타임스탬프 데이터는 전적으로 사용자의 기기에만 저장됩니다. 당사는 다음을 수행하지 않습니다:

- 개인정보 수집
- 원격 서버로 데이터 전송
- 사용 행태 추적
- 위치 정보 접근
- 계정 등록 또는 로그인 요구

## 데이터 저장

Timepaw에서 생성된 모든 데이터는 오직 사용자의 기기에만 로컬로 저장됩니다. 여기에는 다음이 포함됩니다:

- 타임스탬프
- 추가한 메모나 레이블
- 앱 설정 및 환경설정

**사용자의 데이터는 절대 기기 밖으로 전송되지 않습니다.** 단, 사용자가 기기의 기본 공유 기능을 사용하여 명시적으로 내보내기나 공유를 선택한 경우는 예외입니다.

## 제3자 서비스

당사는 다음을 포함한 어떠한 제3자 서비스도 사용하지 않습니다:

- 분석 서비스
- 광고 네트워크
- 크래시 리포팅 도구
- 클라우드 저장소 서비스

## 데이터 보안

모든 데이터가 사용자의 기기에 남아있으므로:

- 데이터는 기기의 기본 보안 기능으로 보호됩니다
- 기기 암호화 및 화면 잠금 기능 사용을 권장합니다
- 데이터 백업 및 복원은 기기의 백업 설정(예: iCloud, 로컬 백업)을 따릅니다

## 아동의 개인정보 보호

Timepaw는 만 14세 미만의 아동을 포함하여 누구로부터도 정보를 수집하지 않습니다. 모든 연령대의 사용자가 안전하게 사용할 수 있습니다.

## 사용자의 권리

당사는 어떠한 데이터도 수집하지 않으므로, 다음과 관련하여 당사가 처리할 개인정보가 없습니다:

- 열람
- 정정
- 삭제
- 이동
- 제3자 공유

모든 데이터는 사용자의 기기에서 사용자의 통제 하에 있습니다.

## 데이터 삭제

사용자는 언제든지 다음 방법으로 모든 앱 데이터를 삭제할 수 있습니다:

- 앱 내에서 개별 타임스탬프 삭제
- 기기에서 앱 삭제(제거)

이렇게 하면 로컬에 저장된 모든 데이터가 영구적으로 제거됩니다.

## 개인정보처리방침의 변경

당사는 때때로 본 개인정보처리방침을 업데이트할 수 있습니다. 변경 사항은 다음과 같이 알려드립니다:

- 앱 내에 새로운 개인정보처리방침 게시
- 본 방침 상단의 "최종 수정일" 업데이트

변경 후 앱을 계속 사용하는 것은 업데이트된 방침에 동의하는 것으로 간주됩니다.
